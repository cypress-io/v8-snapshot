buildDependencyMap

  • buildDependencyMap(inputs: {} & Record<string, { bytes: number; fileInfo: { fullPath: string }; imports: { kind: "require-call"; path: string }[] }>): Map<string, DependencyNode>
    • Defined in src/meta/dependency-map.ts:31

    Builds a dependency map from the inputs of the esbuild metadata.

    This map is embedded into the snapshot in order to allow quickly determine direct and transitive dependencies of a module.

    The map's keys are the module keys. The map's values are DependencyNodes.

    This dependency map is queried when modules load in order to determine if they can be loaded from the cache or if they need to be refreshed to respect Node.js module cache modifications.
