Class SnapshotGenerator

The snapshot generator provides the top level API to create a snapshot script and then convert that into a snapshot binary and finally install it to the correct location.

NOTE: that most fields are directly derived from GenerationOpts and you should refer to the documentation there for more info.

installSnapshot

  • installSnapshot(): void
    • Defined in src/snapshot-generator.ts:579

    Calling this function will first back up the existing electron snapshot unless it was previously backed up. This allows to always revert back to a version of the app without any modified snapshot binary, see uninstallSnapshot.

    Then it will move the snapshot bin into the correct location such that when electron starts up it will load it.

    Returns void

makeSnapshot

  • makeSnapshot(): Promise<null | { v8ContextFile: string }>
    • Defined in src/snapshot-generator.ts:518

    This will call the mksnapshot command feeding it the snapshot script previously created via createScript which needs to be invoked before running this function.

    The resulting snapshot binary is written to this.snapshotBinPath and needs to be moved to the correct location by calling installSnapshot.

    Returns Promise<null | { v8ContextFile: string }>
