Class WarningsProcessor

Processes raw Warnings emitted from the snapshot bundler.

Determines what WarningConsequence for the module where the it originated the warning has as well as preparing it for printing to the console.

process

  • process(warnings: Message[], hist: WarningsProcessHistory): ProcessedWarning[]
    • Defined in src/doctor/warnings-processor.ts:130

    Takes a list of warnings as well as a history of warnings processed previously. It then processes each warning in a ProcessedWarning, filters out the ones already processed previously and returns the remaining ones.

    The consequence of the warning is determined by parsing its Warning text via regexes and string comparisons and thus is highly dependent on the format in which the snapshot bundler emits them.

    Parameters

    • warnings: Message[]

      warnings emitted by the snapshot script bundler

    • hist: WarningsProcessHistory

      history of previously processed warnings

    Returns ProcessedWarning[]
